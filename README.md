
# Fork And Knife

This is the website that shows the recipes of all available chefs. The website is exclusively dedicated to Italian cuisine. 


## Features

- Register with firbase
- Login with firebase
- Blog
- Service
- Chef Detail
- Recipe
- Footer
- Authentication
- View Recipes
- Chef Recipes
- Favorite button
- Toastify
- Spinner
- 404 page
- Environment variable to hide the Firebase config keys
- Active Route
- Protected page
